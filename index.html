What Is CSS Grid

CSS Grid is a layout system that consists of both rows and columns, allowing the user to be able to manipulate a part of their site in 2 dimensions. This makes styling websites a lot easier, before grid was created a lot of developers used the table element to mimic what a grid can do. This was a huge pain for most people, so when grid was created developers and designers alike rejoiced! CSS Grid was introduced in 2012 by Bert Bos, and his associate HÃ¥kon Wium Lie. Grid started as a very simple way to show content on a very small screen. When grid was first introduced developers/designers all over the world looked at it as a revolutionary way to change the layout of pages, and to this day is used in hundreds of websites all over the Internet.

HTML

<div class="grid">
    <div class="col-span-2">1</div>
    <div class="row-span-2">2</div>
    <div>3</div>
    <div>4</div>
    <div>5</div>
    <div class="col-span-2">6</div>
</div>

CSS

.grid {
    display: grid;
    grid-gap: 0.5rem;
    grid-template-columns: repeat(3, auto);
    padding: 0.5rem;
    height: 100%;
    box-sizing: border-box;
}

How to Build a Grid Container

To make a Grid Container you must set the display property to grid or inline-grid. Grid containers are made up of grid items placed inside of columns and rows. To achieve the example below, first create a div with a class of "grid-container." Next, create additional div elements for each column you want your grid to have. Direct child elements of the grid container automatically become grid items. Lastly, add custom CSS for how you would like your grid to display on the website.

HTML

<div class="grid-container">
    <div>1</div>
    <div>2</div>
    <div>3</div>
    <div>4</div>
</div>

CSS

.grid-container {
    display: grid;
    grid-template-columns: repeat(4, auto);
    grid-gap: 0.5rem;
    padding: 0.5rem;
}

How to Align Items in a Grid

grid-row

The grid-row property is shorthand for grid-row-start and grid-row-end which determine the start and end position of a grid item across rows. This is accomplished by stating the row start position before the / and the row end position after.

HTML

<div class="grid">
    <div class="col-row-2">1</div>
    <div>2</div>
    <div>3</div>
    <div class="col-row-1-end-3">4</div>
</div>

CSS

.col-row-2 {
    grid-row: 2;
}
.col-row-1-end-3 {
    grid-row: 1 / 3;
}

grid-column

The grid-column property is shorthand for grid-column-start and grid-column-end which determine the start and end position of a grid item across columns. This is accomplished by stating the column start position before the / and the column end position after.

HTML

<div class="grid">
    <div class="col-start-3">1</div>
    <div>2</div>
    <div>3</div>
    <div class="col-start-2-end-4">4</div>
</div>

CSS

.col-start-3 {
    grid-column: 3;
}
.col-start-2-end-4 {
    grid-column: 2 / 4;
}

grid-area

The grid-area property is shorthand for positioning items across both the rows and columns of a grid. It first starts with the grid-row-start and is followed by the grid-column-start, grid-row-end, and grid-column-end with slashes in between.

HTML

<div class="grid">
    <div class="square-right">1</div>
    <div>2</div>
    <div>3</div>
    <div>4</div>
    <div>5</div>
    <div>6</div>
</div>

CSS

.square-right {
    grid-area: 2 / 2 / 4 / 4;
}

grid-template-areas

The grid-template-areas property is just like grid-area but the position of items are based off of specified names rather than values. To use this method of positioning you must declare the name of each item with the grid-area property. Then you must declare the position of the item using its name with the grid-template-areas property.

HTML

<div class="grid">
    <div id="one">1</div>
    <div id="two">2</div>
    <div id="three">3</div>
    <div id="four">4</div>
</div>

CSS

.grid-template-areas {
    grid-template-areas: 
        "one  two   two "
        "one  three .   "
        ".    three .   "
        "four four  four";
}
